# **App Name**: SenseMapper

## Core Features:

- Sensory Layer Toggle: Enable or disable individual sensory layers (Touch, Vestibular, Proprioception, Vision, Hearing, Smell, Taste) for focused map views.
- Interactive Markers: Place icons representing sensory inputs (e.g., sound icon for loud noise).
- Zone Definition: Draw rectangular zones to highlight areas with specific sensory qualities.
- Annotation Editor: Add and edit descriptive annotations to markers and zones.
- Generate Insights Tool: Use an AI tool to automatically generate summaries of sensory considerations in selected areas.

## Style Guidelines:

- Primary color: Deep indigo (#4B0082) to represent the complexity of sensory experiences.
- Background color: Very light lavender (#E6E6FF) for a calming, unobtrusive backdrop.
- Accent color: Soft violet (#8A2BE2) for interactive elements.
- Body and headline font: 'PT Sans', a humanist sans-serif for a balance of modern and warm feel.
- Custom icons for each sensory layer (Touch, Vestibular, Proprioception, Vision, Hearing, Smell, Taste).
- Sidebar for layer control and annotation tools; map area for sensory mapping.
- Smooth transitions for layer toggling and marker placement.
